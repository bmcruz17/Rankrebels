# Rank Rebels — apparel / print artwork

Print-ready artwork for the Rank Rebels flag logo, rebuilt as **true vector** from the
brand mark. Hand these to any screen printer or embroiderer.

- **`.svg` = the master.** Vector — scales to any size with no quality loss. Send these first.
- **`.png` = transparent 300&nbsp;dpi backup** for shops that only take raster.
- Start here: **`rr-artwork-sheet.png`** — one page showing every file, placement, sizes, and ink/thread specs.

## Colours
| | Hex | Notes |
|---|---|---|
| Brand green | `#1aa251` | ≈ Pantone **355 C / 7739 C** — match to the printer's physical swatch |
| Black ink | `#111111` | standard black |
| White ink | `#ffffff` | standard white |

## Files

**Full-colour marks (green monogram)**
- `rr-flag-color-light.svg/.png` — black flag + green RR → **white / light garments**
- `rr-flag-color-dark.svg/.png` — white flag + green RR → **black / dark garments**

**One-colour marks** (single ink; the RR monogram knocks out to the shirt colour — cheapest to print)
- `rr-flag-1c-black.svg/.png` — black ink, for light/coloured shirts
- `rr-flag-1c-white.svg/.png` — white ink, for dark/coloured shirts
- `rr-flag-1c-green.svg/.png` — single green ink option

**Layouts**
- `rr-front-breast-light/dark` — left-breast mark only, **no URL**, ~3.5–4″ wide
- `rr-back-light/dark` — full back: flag → **RANK REBELS** → tagline → **rankrebels.ai**, ~11″ wide

## Placement
- **Front:** left breast, mark only, on the pocket line, **3.5–4″** wide.
- **Back:** centred under the collar, **10.5–12″** wide.

## Tagline
Current: **RANK HIGHER. REBEL HARDER.** (baked into the back layouts). Easy to swap — say the word.

## Embroidery
Send the full-colour SVG (`-color-light` / `-color-dark`); the shop digitizes it to a stitch
file (.DST/.PES). Thread: green ≈ **Madeira 1051 / Isacord 5411**; flag in **white** on dark
shirts, **black** on light. Keep the mark **≥ 3.5″** (chest) / **≥ 8″** (back) so the centre
arrow and flagpole hold their detail.
